# Film Directory Archiver
> Generate a static file verison of the BiM Film Directory

## Requirements

- Python 3.9+
- Poetry 1.1.0+

### Required Secrets
- API_DOMAIN
- DIST_PATH
- SITE_DOMAIN
- DOCKER_SECRETS_PATH (in env)


## Usage
- `poetry install`
- `poetry shell`
- `python ./archiver.py`
